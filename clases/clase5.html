📖 ¿Qué es un Arreglo en Python?

En Python puro, las listas se comportan como arreglos dinámicos. Un arreglo tradicional, como en otros lenguajes (C, Java), es una estructura ordenada de elementos del mismo tipo.

En programación científica o industrial se recomienda usar **NumPy** para arreglos más eficientes, pero en este curso inicial trabajaremos listas como arreglos.

📖 Propiedades de los Arreglos (Listas en Python)

  • Ordenados: los elementos tienen una posición fija.
  • Indexables: se puede acceder mediante índices.
  • Mutables: los elementos pueden ser cambiados.
  • No necesariamente homogéneos: en listas básicas.

📖 Operaciones Básicas sobre Arreglos

🔹 Acceso a elementos:


print(temperaturas[2])  # 26

🔹 Modificar elementos:


temperaturas[1] = 23

🔹 Agregar elementos:


temperaturas.append(28)

🔹 Eliminar elementos:


temperaturas.pop(0)  # Elimina el primer elemento

📖 Indexación en Arreglos

Los arreglos en Python permiten el acceso mediante índices positivos y negativos.

Elemento Índice positivo Índice negativo
22 0 -5
24 (modificado a 23) 1 -4
26 2 -3
27 3 -2
25 4 -1

📖 Slicing (Subarreglos)

Slicing permite acceder a una sección del arreglo:

🔹 Ejemplos:


print(temperaturas[1:4])  # [23, 26, 27]
print(temperaturas[:3])   # [23, 26, 27]
print(temperaturas[2:])   # [26, 27, 25]

🔹 Slicing avanzado (con pasos):


print(temperaturas[::2])  # [23, 27]

📖 Aplicaciones Industriales de Arreglos

  • Registrar temperaturas diarias de hornos industriales.
  • Almacenar conteos de producción por turno.
  • Control de inventario de repuestos clasificados por tipo.

🔹 Ejemplo Industrial:


produccion_turnos = [120, 135, 110, 145]

# Promedio de producción
promedio = sum(produccion_turnos) / len(produccion_turnos)
print("Promedio de producción diaria:", promedio)

🧪 APE - Actividad Práctica Experimental (en clase)

🔹 Actividad:

Crear un arreglo (lista) de las temperaturas registradas en una planta durante 5 días.

Problema:

  • Registrar las temperaturas.
  • Calcular la temperatura promedio.
  • Identificar si alguna temperatura supera los 30°C.

Código base:


temperaturas = [28, 29, 31, 27, 30]

promedio = sum(temperaturas) / len(temperaturas)
print("Temperatura promedio:", promedio)

for temp in temperaturas:
    if temp > 30:
        print("Alerta: temperatura alta detectada:", temp)

📝 AA - Actividad Autónoma (para casa)

🔹 Actividad:

Registrar la producción de una línea industrial durante 6 turnos.

Problema:

  • Registrar producción en lista.
  • Calcular producción total.
  • Calcular producción promedio.
  • Identificar turnos con producción menor a 100 unidades.

Código sugerido:


produccion_turnos = [95, 105, 120, 98, 130, 102]

produccion_total = sum(produccion_turnos)
promedio = produccion_total / len(produccion_turnos)

print("Producción total:", produccion_total)
print("Promedio de producción:", promedio)

for turno, produccion in enumerate(produccion_turnos, start=1):
    if produccion < 100:
        print("Turno", turno, "producción baja:", produccion)
