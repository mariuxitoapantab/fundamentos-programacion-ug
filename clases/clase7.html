📚 Repaso de Contenidos – Unidad 1

🔹 1.2. Interpretadores y Compiladores

Interpretador: ejecuta el código línea por línea (ej: Python).

Compilador: traduce todo el código fuente a código máquina antes de ejecutarlo (ej: C++).

🔹 1.2. Ambientes de Programación

Programas que facilitan el desarrollo de software (IDLE, VSCode, PyCharm, Psint).

Instalación y uso inicial de Psint para algoritmos básicos.

🔹 1.2. Conceptos y Propiedades de los Algoritmos

Definición de algoritmo: secuencia de pasos lógicos para resolver un problema.

Propiedades esenciales: finitud, precisión, entrada, salida, eficacia.

🔹 1.3. Variables, Constantes y Tipos de Datos

Variable: espacio de memoria para almacenar datos que pueden cambiar.

Constante: valor que no debe modificarse durante la ejecución.

Tipos de datos: int, float, str, bool.

🔹 1.3. Operadores Lógicos y Relacionales

Relacionales: ==, !=, <, >, <=, >=.

Lógicos: and, or, not.

🔹 1.3. Manejo de Entrada-Salida

input(): para capturar datos del usuario.

print(): para mostrar información en pantalla.

📚 Repaso de Contenidos – Unidad 2

🔹 2.1. Estructuras de Control Condicionales

if, if-else, if-elif-else: permiten tomar decisiones basadas en condiciones.

🔹 2.1. Estructuras de Control Iterativas

for, while: permiten repetir bloques de código múltiples veces.

🔹 2.1. Sentencias Anidadas

if dentro de otro if: se usan estructuras dentro de otras para tomar decisiones complejas.

🔹 2.1. Switch (match-case)

Implementado en Python 3.10+ como match-case. Alternativa organizada a múltiples if-elif-else.

🔹 2.2. Listas: Propiedades, Operaciones, Indexación y Slicing

Listas son colecciones ordenadas y mutables.

Indexación positiva y negativa.

Sublistas (slicing): crear sublistas a partir de listas.

🔹 2.2. Manejo de Cadenas como Listas

Los str pueden ser tratados como listas de caracteres.

🔹 2.3. Arreglos: Propiedades, Operaciones, Indexación y Slicing

Diferencia entre listas y arreglos.

Arreglos reales requieren NumPy para operaciones avanzadas.

🔹 2.4. Operaciones Aritméticas, Estadísticas y de Ordenamiento

Suma, promedio, mínimo, máximo.

Ordenamiento ascendente y descendente.

🔹 2.4. Introducción al Broadcasting

Listas: simulación con comprensión de listas.

Arreglos NumPy: Broadcasting real y automático.
