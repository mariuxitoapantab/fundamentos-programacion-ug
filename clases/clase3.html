Clase 3 - Condicionales, Switch, Iterativas

Objetivos de Aprendizaje

  • Aplicar estructuras condicionales (`if`, `if-else`, `if-elif-else`).
  • Implementar el control múltiple usando `match-case` (switch).
  • Utilizar estructuras iterativas (`for`, `while`).
  • Manejar instrucciones especiales: `break`, `continue`, `pass`.

Introducción

Nota: En esta clase también se aplica la Lección de la Unidad 1 correspondiente a los contenidos desarrollados en las clases anteriores.

Contenido

Estructuras Condicionales


temperatura = 30

if temperatura > 25:
    print("Activar ventilación industrial")
else:
    print("Sistema normal")

Switch con match-case (Python 3.10+)


opcion = int(input("Seleccione opción (1-3): "))

match opcion:
    case 1:
        print("Registrar producción")
    case 2:
        print("Consultar inventario")
    case 3:
        print("Salir del sistema")
    case _:
        print("Opción inválida")

Estructuras Iterativas


for i in range(5):
    print("Operario número:", i+1)

contador = 0
while contador < 5:
    print("Mantenimiento máquina:", contador+1)
    contador += 1

Instrucciones break, continue y pass


for pieza in range(1, 6):
    if pieza == 4:
        break
    print("Pieza procesada:", pieza)

for pieza in range(1, 6):
    if pieza == 2:
        continue
    print("Pieza verificada:", pieza)

Resumen de la Clase

  • Control del flujo de ejecución usando condicionales e iteraciones.
  • Implementación eficiente con `match-case` en Python 3.10+.
  • Optimización de ciclos con `break`, `continue` y `pass`.
