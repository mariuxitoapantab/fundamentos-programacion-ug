📖 Condicionales en Python

🔹 if simple


temperatura = 32

if temperatura > 30:
    print("Encender ventiladores industriales.")

🔹 if-else


stock_actual = 45
stock_minimo = 50

if stock_actual >= stock_minimo:
    print("Stock suficiente.")
else:
    print("Solicitar reposición de inventario.")

🔹 if-elif-else


eficiencia = 78

if eficiencia >= 90:
    print("Excelente desempeño.")
elif eficiencia >= 75:
    print("Desempeño aceptable.")
else:
    print("Desempeño bajo, requiere mejora.")

📖 Switch - match-case en Python 3.10+

Python implementa switch mediante match-case, más organizado que muchos if-elif-else seguidos.

🔹 Ejemplo:


opcion = int(input("Seleccione opción (1-Registrar, 2-Reportes, 3-Salir): "))

match opcion:
    case 1:
        print("Registrando producción...")
    case 2:
        print("Mostrando reportes...")
    case 3:
        print("Saliendo del sistema...")
    case _:
        print("Opción inválida.")

📌 Nota: Solo disponible en Python 3.10+.

📖 Estructuras Iterativas

🔹 while


contador = 1
while contador <= 5:
    print("Producto inspeccionado:", contador)
    contador += 1

🔹 for


for operario in range(1, 6):
    print("Asignar turno a operario:", operario)

📖 Sentencias Anidadas

Usar un if dentro de otro if.


temperatura = int(input("Ingrese temperatura: "))

if temperatura > 30:
    if temperatura > 40:
        print("Alerta: sobrecalentamiento")
    else:
        print("Precaución: temperatura alta")
else:
    print("Temperatura normal")

📖 Break, Continue y Pass

🔹 break: Finaliza el bucle inmediatamente.


for pieza in range(1, 10):
    if pieza == 5:
        break
    print("Pieza procesada:", pieza)

🔹 continue: Salta el resto de instrucciones del ciclo.


for pieza in range(1, 6):
    if pieza == 3:
        continue
    print("Pieza:", pieza)

🔹 pass: Deja un bloque vacío sin errores.


for i in range(5):
    pass  # Se completará más tarde

  • Horas de funcionamiento > 8
  • Temperatura > 70°C

Código sugerido:


horas = int(input("Ingrese horas de funcionamiento: "))

if horas > 8 or temperatura > 70:
    print("Detener máquina para mantenimiento")
else:
    print("Operación normal")

tipo_maquina = int(input("Tipo de máquina (1-Motor, 2-Generador, 3-Compresor): "))

match tipo_maquina:
    case 1:
        print("Motor seleccionado")
    case 2:
        print("Generador seleccionado")
    case 3:
        print("Compresor seleccionado")
    case _:
        print("Tipo desconocido")
