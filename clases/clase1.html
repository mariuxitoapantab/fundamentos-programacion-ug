Interpretadores y Compiladores

Un interpretador ejecuta el código línea por línea. Un compilador traduce todo el código fuente a lenguaje máquina antes de ejecutarlo.

Tabla Comparativa

Tipo Proceso
Interpretador Traduce línea por línea y ejecuta al momento.
Compilador Traduce todo el programa antes de ejecutarlo.

Ambientes de Programación

Un ambiente de programación es un conjunto de herramientas que permite escribir, depurar y ejecutar programas.

  • Psint: Ideal para algoritmos básicos. Descargar desde el sitio oficial.
  • IDLE (Python): Ambiente básico que viene con Python.
  • VSCode: Editor de código muy popular para múltiples lenguajes.

Conceptos de Algoritmos

Un algoritmo es una serie de pasos finitos y ordenados que resuelven un problema específico.

Propiedades de los Algoritmos:

  • Finitud: Termina en un número limitado de pasos.
  • Precisión: Cada paso es claro y específico.
  • Entrada: Requiere datos de entrada.
  • Salida: Produce uno o más resultados.
  • Eficacia: Cada paso debe ser realizable en tiempo finito.

Ejemplo de un Algoritmo:

  1. Iniciar.
  2. Leer dos números.
  3. Sumar los números.
  4. Mostrar el resultado.
  5. Finalizar.

Instalación y Primer Programa en Psint

  • Visitar el sitio oficial: https://www.pseint.org/
  • Descargar el instalador correspondiente a tu sistema operativo.
  • Instalar y abrir Psint.

Primer ejercicio en Psint:


// Programa que suma dos números
Algoritmo SumaDosNumeros
    Definir num1, num2, suma Como Entero
    Escribir "Ingrese primer número:"
    Leer num1
    Escribir "Ingrese segundo número:"
    Leer num2
    suma <- num1 + num2
    Escribir "La suma es:", suma
FinAlgoritmo
