Clase 1 – Interpretadores, Compiladores, Ambientes de Programación, Conceptos de Algoritmos

Objetivos de Aprendizaje

  • Distinguir entre interpretadores y compiladores.
  • Conocer los ambientes de programación básicos.
  • Comprender los conceptos fundamentales de los algoritmos.
  • Realizar primeros ejercicios en Psint.

Interpretadores y Compiladores

Interpretador: ejecuta el código línea por línea. Ejemplos: Python, JavaScript.

Compilador: traduce todo el código fuente a lenguaje máquina antes de ejecutarlo. Ejemplos: C, C++.

Ejemplo visual:

  • Interpretador: Traduce línea por línea y ejecuta al momento.
  • Compilador: Traduce todo el programa antes de ejecutarlo.

Ambientes de Programación

Un ambiente de programación es un conjunto de herramientas que permite escribir, depurar y ejecutar programas.

  • Psint: Ideal para algoritmos básicos. Sitio oficial de Psint
  • IDLE: Entorno integrado de Python.
  • VSCode: Editor de código moderno y flexible.

Conceptos de Algoritmos

Un algoritmo es un conjunto de pasos finitos y ordenados que permiten resolver un problema.

Propiedades de los algoritmos:

  • Finitud: Termina en un número limitado de pasos.
  • Precisión: Cada paso está claramente definido.
  • Entrada: Recibe datos iniciales.
  • Salida: Produce un resultado.
  • Eficacia: Cada paso es sencillo de realizar.

Ejemplo de un algoritmo básico:

  1. Iniciar.
  2. Leer dos números.
  3. Sumarlos.
  4. Mostrar el resultado.
  5. Fin.

Instalación y Primer Programa en Psint

Para instalar Psint:

  • Ir a: https://www.pseint.org/
  • Descargar el instalador correspondiente.
  • Instalar y abrir el programa.

Primer ejercicio en Psint:


// Este programa suma dos números
Algoritmo SumaDosNumeros
    Definir num1, num2, suma Como Entero
    Escribir "Ingrese primer número:"
    Leer num1
    Escribir "Ingrese segundo número:"
    Leer num2
    suma <- num1 + num2
    Escribir "La suma es: ", suma
FinAlgoritmo
