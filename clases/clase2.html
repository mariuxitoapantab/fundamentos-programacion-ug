Clase 2 - Variables, Constantes y Tipos de Datos

Objetivos de Aprendizaje

  • Definir variables y constantes en Python.
  • Utilizar operadores lógicos y relacionales.
  • Realizar conversiones entre tipos de datos.
  • Implementar entrada y salida estándar con input() y print().

Contenido

¿Qué es una Variable?

Una variable es un espacio de memoria reservado para almacenar datos que pueden cambiar durante la ejecución.

produccion_diaria = 480

¿Qué es una Constante?

Una constante es un espacio reservado cuyo valor no debe cambiar durante el programa. Se escribe en mayúsculas.

PI = 3.1416

Tipos de Datos en Python

  • int: enteros (ej. 50)
  • float: decimales (ej. 3.14)
  • str: textos (ej. "Producción")
  • bool: booleanos (True/False)

Operadores Relacionales y Lógicos

  • Relacionales: ==, !=, >, <, >=, <=
  • Lógicos: and, or, not

Conversión entre Tipos de Datos


entrada = "100"
numero = int(entrada)

cantidad = 200
texto = str(cantidad)

Entrada y Salida de Datos


nombre = input("Ingrese su nombre: ")
print("Bienvenido,", nombre)

Resumen de la Clase

  • Variables y constantes controlan los datos en los programas.
  • Conversión de tipos es fundamental para interactuar con usuarios.
  • Entrada y salida permiten comunicación entre el programa y el usuario.
